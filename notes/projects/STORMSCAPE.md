### STORMSCAPE

- 그 당시 즐겨 했던 **게임의 데이터**를 api 로 가져와서 **효과적으로 시각화**한 것을 공유하기 위해 만듦 
- 각 게임 캐릭터의 최근 승률 (모든 경우, 특정 영웅이 같은 팀 혹은 상대 팀일 때, 특정 아이템 선택했을 때 등), 각 게임 캐릭터 플레이스타일 데이터 정돈해서 시각화
- **Preact**으로 빌드 과정없이 작성 
- 모바일로 사용할 것을 가정하고 디자인
- 현재 api 비용 관계로 배포 중지
